Counter
Instructions
  1. Click on the Decrease button if you wish to reduce the number by one.
  2. Click on the Increase button if you wish to increase the number by one.
  3. Click on the Random button to generate a number between -1,000 and 1,000.
  4. Click on the Prime button if you wish to find out whether the number displayed is a prime number.
  5. Click on the Reset button to change the number displayed to 0.
  6. Use the Up arrow key to increase the number displayed.
  7. Use the Down arrow key to decrease the number displayed.
